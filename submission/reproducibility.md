# Reproducibility instructions

Public artifact: `github.com/appleweiping/inr-aliasing-limits` (MIT, code only). ICASSP 2027
is single-anonymous, so the public repo is compatible with submission. This file mirrors
the repo README's reproduction section for the submission package.

## CPU reproduction (all theory + the design contribution)

```bash
git clone https://github.com/appleweiping/inr-aliasing-limits
cd inr-aliasing-limits
uv venv --python 3.11 && source .venv/bin/activate     # Windows: .venv/Scripts/activate
uv pip install -r requirements.lock && uv pip install -e . --no-deps
python -m pytest -q                    # 39 tests: theorems + design + certificate
python experiments/run_all.py          # regenerates E1-E3, E6 (design), E5 part A
```

Deterministic (fixed seeds). Outputs land in `results/*.json` (each with a `_meta`
environment stamp) and `results/figures/*.png`. The manuscript figures are exactly these
PNGs; the AliasGuard numbers in the manuscript are generated by `paper/sync_macros.py` from
`results/aliasguard.json` (no hand-entered numbers).

## Full reproduction (adds the trained-network extension E4 / E5-B)

Requires torch + a CUDA GPU:
```bash
python experiments/run_all.py --full
```
GPU training sets deterministic algorithms where available; results are statistically (not
bitwise) reproducible across CUDA/toolkit versions, as stated in the README.

## Building the paper

```bash
cd paper
python sync_macros.py                  # macros_ag.tex <- results/aliasguard.json
pdflatex main && bibtex main && pdflatex main && pdflatex main
pdflatex supplement && pdflatex supplement
```

## Environment used for the released results

See `submission/environment.txt` (Python/NumPy/SciPy/torch/CUDA/GPU versions) and the
`_meta` block inside each `results/*.json`. Dependency versions are pinned in
`requirements.lock`.

## What maps to what

`submission/results-manifest.txt` lists every released result file with a SHA-256 checksum
and the script that produces it. `docs/claim-ledger.md` maps every paper claim to its
proof, code, JSON, and figure.
